## 项目代号
项目代号：**Galileo**  
**伽利略・伽利雷**（1564年2月15日～1642年1月8日）享年77岁，原名（Galileo di Vincenzo Bonaulti de Galilei）是意大利天文学家、物理学家和工程师、欧洲近代自然科学的创始人。伽利略被称为“观测天文学之父”、“现代物理学之父”、“科学方法之父”、“现代科学之父”。
该项目以Galileo作为项目代号，以此纪念这位伟大的天文学家。
## 项目简介
**Galileo** 目前的定位是一款**面向中国机械工业行业**的、***开源***的、采用**垂直知识图谱**+**搜索引擎**技术研发的应用平台。  
该项目工程目前包括：**数据采集（Scavenger）**、**垂直知识图谱+搜索引擎（Galileo）**、 **运维管理(Drucker)** 三块主营业务。
## 技术路线
**Galileo**的基座采用目前主流的、基于服务网格的下一代微服务框架**Dapr**进行统一开发的。通过Dapr提供的服务调用方式打通数据采集服务、搜索引擎服务、知识图谱服务等。
